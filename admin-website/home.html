Project Overview


Project PlumPantry is a culinary website that allows visitors to view and look up recipes with advanced search options, such as selecting specific ingredients or choosing broad categories of meal types or cuisines. Registered users can also create, edit, publish, and/or delete their own recipes, which are shared on the homepage. The website will have a recipe editor where registered users can add instructions, list ingredients, and upload a cover image for their recipe.

Server-side Tools and Technologies:

  • Spring Boot 2+: Framework for building Java-based enterprise applications.
  • Spring Data JPA (Hibernate): Tool for managing relational data in Java applications.
  • Maven 3.2+: Build automation tool used for project management and dependency management.
  • JDK 1.8: Java Development Kit version 1.8.
  • Embedded Tomcat 8.5+: Servlet container for hosting Java-based web applications.
  • MongoDB - non-relational (noSQL) database management system storing recipe, ingredient, and user data in pages.

Client-side Tools and Technologies:

  • React: JavaScript library for building user interfaces.
  • Modern JavaScript (ES6): JavaScript language standard used for client-side scripting.
  • NodeJS and NPM: JavaScript runtime and package manager, respectively.
  • Create React App CLI: Command-line tool for creating React applications.
  • Bootstrap 4.5: Front-end framework for developing responsive and mobile-first websites.
  • Axios HTTP Library: JavaScript library for making HTTP requests.
